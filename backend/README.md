## *Auth*
Authentication of Users leveraging password hashing and JWT based authentication and authorization

### End-point: Signup
Register/Sign Up a new user to the platform and store the encrypted password
#### Method: POST
>```
>/api/auth/signup
>```
#### Body (**raw**)

```json
{
    "username":"testuser1232",
    "email":"testuser1232@test.com",
    "password":"test"
}
```

## *Group Expense*

### End-point: Create Group Expense
Create an expense in a group with all the splits
#### Method: POST
>```
>/api/group/expense/:groupId
>```
#### Body (**raw**)

```json
{
    "amount":"500", 
    "description":"breakfast",
    "type":"equal",
    "date":"2024-07-29",
    "borrowers": [
    { "borrowerId": 4, "percentage": 50 },
    { "borrowerId": 6, "percentage": 40 }

}
```
